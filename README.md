## How to use

- Start with each chapter’s `01-notes.md`.
- Then do `02-solved-questions.md` and rewrite the answers in your own words.
- For numericals: copy the **method** first, then solve fresh values.

## Exam strategy (recommended)

- First pass (fast): read `01-notes.md` and memorize the **definitions + diagrams**.
- Second pass (scoring): do all `02-solved-questions.md` without looking, then compare.
- Build a 1-page sheet per chapter: formulas, steps, and 5 “most asked” questions.
- Before exam: revise Chapters 5–12 extra carefully (they contain most numericals/algorithms).
- For weak areas: re-solve the same numericals with new values until your method is automatic.
